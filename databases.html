Databases

The following data resources are hosted by Elixir Ireland associated researchers:

Tools Descriptions
DARNED A database of human RNA editing. It has not been updated for a while, but is still surprisingly popular.
Maintainer: Pavel Baranov
GWIPS-viz A genome browser for the visualisation of public ribosome profiling data aligned to genomes.
Maintainer: Pavel Baranov
Trips-viz A computational environment for the analysis and visualisation of public and user-generated ribosome profiling data mapped to transcriptomes.
Maintainer: Pavel Baranov
CancerGD: analysis of genetic dependencies in cancer Database of cancer genetic dependencies identified in CRISPR/RNAi screens.
Maintainer: Colm Ryan
ParalogSL Database of predicted synthetic lethal interactions between paralogs in cancer.
Maintainer: Colm Ryan
