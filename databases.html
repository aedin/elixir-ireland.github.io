Databases

The following data resources are hosted by Elixir Ireland associated researchers:

DARNED
A database of human RNA editing. It has not been updated for a while, but is still surprisingly popular
GWIPS-viz
A genome browser for the visualisation of public ribosome profiling data aligned to genomes
Trips-viz
Source code (C++). Software for model selection during phylogenetic analysis of nucleotide and amino acid sequences. ModelFinder is embedded in UQ-TREE and IQ-TREE 2 and can only be access by downloading these programs.
CancerGD: analysis of genetic dependencies in cancer
Database of cancer genetic dependencies identified in CRISPR/RNAi screens
ParalogSL
Database of predicted synthetic lethal interactions between paralogs in cancer
